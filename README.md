# Clothen - 前端電商網站

![Clothen Logo](./img/Clothen.jpg)

## 專案介紹
Clothen 是一個模擬前端服飾電商網站，提供商品瀏覽、分類、搜尋、購物車、輪播圖、使用者回饋及退換貨流程等功能。  
專案主要目標是練習 **HTML、CSS、JavaScript** 的前端開發，以及前端互動功能整合。

## 功能特色
- 商品展示與分類瀏覽（Women / Men / Kids）
- 輪播圖展示季節商品
- 搜尋商品功能
- 購物車功能：
  - 加入商品
  - 移除商品
  - 小計計算
- TOP 回到頁面頂端按鈕
- Footer 功能：
  - 關於 Clothen
  - 線上退換貨流程
  - 意見回饋表單
  - 社群連結（Facebook / Instagram / Line）

## 專案結構
Clothen/
│-- CSS/                 # 樣式檔案
│-- JavaScript/          # 功能模組
│-- img/                 # 圖片資源
│-- index.html           # 首頁
│-- about.html           # 關於我們頁面
│-- categorySidebar*.html# 商品分類頁
│-- csReturnFlow.html     # 線上退換貨頁面
│-- line.html            # Line 連結頁面
│-- userFeedback.html    # 使用者回饋頁面

## 技術堆疊
- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome 圖示庫

## 安裝與使用
1. Clone 專案到本地：
```
git clone https://github.com/JoeCode08888/clothen-website.git
```

2. 進入專案資料夾：
```
cd clothen-website
```

3. 打開 `index.html`，即可在瀏覽器預覽網站：
```
start index.html  # Windows
open index.html   # Mac
```
